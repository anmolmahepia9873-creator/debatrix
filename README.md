# Debatrix — AI · NLP · Debate Engine

A self-built AI-powered decision engine that takes any problem 
and two competing options and generates a structured debate-style 
analysis to help users make better decisions.

## What it does
- Takes a problem and two options as input
- Generates FOR and AGAINST arguments for each option
- Declares a winner with a confidence score
- Built with predefined domain knowledge — no external API needed

## Tech Stack
- Python
- NLP Logic
- HTML · CSS · JavaScript
